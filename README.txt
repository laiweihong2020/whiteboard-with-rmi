 
If you wish to compile the client or server folders, make sure you have Maven installed in your system.

Build command for client: "mvn package"
Build command for server: "mvn clean compile assembly:single"

If you just wish to run the program, navigate to the target file and execute the jar file
Target file server: server-1.0-jar-with-dependencies.jar
Target file client: client-with-whiteboard-1.0.jar
